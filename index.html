Habilidades

JavaScript

Criação de interfaces interativas com manipulação eficiente do DOM e uso de ES6+ para código moderno. Operações assíncronas, alta performance e fluidez nas aplicações

Node.js

Desenvolvimento de APIs RESTful e gerenciamento de servidores. Pontos fortes: escalabilidade, integração com bancos de dados e otimização de desempenho.

React

Desenvolvimento de interfaces reativas com hooks e componentes funcionais. Pontos fortes: modularidade, gerenciamento eficaz de estado e experiência de usuário fluida.

Figma

Design de interfaces responsivas e prototipagem interativa. Pontos fortes: criação de designs intuitivos, colaboração eficiente e adesão às melhores práticas de UX/UI

Git

Controle de versão com Git, incluindo branching e merging. Pontos fortes: colaboração em equipe, organização do código e gerenciamento de histórico.

npm

Gerenciamento de pacotes e automação de tarefas. Pontos fortes: integração contínua, manutenção de dependências e scripts personalizados para otimização.
